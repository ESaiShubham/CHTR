# Chhattisgarh Travel Explorer 🌟

A modern, responsive web application showcasing the beautiful tourist destinations of Chhattisgarh, India. Built with React, featuring a stunning purple/white design theme and smooth animations.

## ✨ Features

- **Modern Design**: Beautiful purple/white color scheme with gradient effects
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Smooth Animations**: Framer Motion powered animations and transitions
- **Interactive Cards**: Hover effects and modern card designs
- **Search & Filter**: Find destinations by category and search terms
- **Static Data**: No backend required - all data is included locally
- **Fast Performance**: Optimized for speed and user experience

## 🎨 Design Highlights

- **Purple/White Theme**: Elegant color palette with gradient accents
- **Modern Typography**: Inter font family for clean, readable text
- **Card-Based Layout**: Airbnb-inspired destination cards
- **Smooth Transitions**: Hover effects and page transitions
- **Mobile-First**: Responsive design that works on all screen sizes

## 🚀 Quick Start

### Prerequisites
- Node.js (version 14 or higher)
- npm (comes with Node.js)

### Installation

1. **Clone or download** this project to your computer

2. **Run the setup script** (Windows):
   ```bash
   setup.bat
   ```
   
   Or manually install dependencies:
   ```bash
   cd frontend
   npm install
   ```

3. **Start the application**:
   ```bash
   start.bat
   ```
   
   Or manually:
   ```bash
   cd frontend
   npm start
   ```

4. **Open your browser** and go to `http://localhost:3000`

## 🚀 Deploy to Vercel

### Quick Deployment

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Ready for Vercel deployment"
   git push origin main
   ```

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign in with GitHub
   - Click "New Project"
   - Import your repository
   - Click "Deploy"

Your app will be live at `https://your-project-name.vercel.app`

### Detailed Deployment Guide

For step-by-step instructions, see [DEPLOYMENT.md](./DEPLOYMENT.md)

## 🌟 Why Frontend-Only?

This application is designed as a **frontend-only** solution for several benefits:

- **Easy Setup**: No backend configuration required
- **Fast Deployment**: Can be deployed to any static hosting service
- **Low Maintenance**: No server management needed
- **Cost Effective**: No hosting costs for backend services
- **Reliability**: No database dependencies or API failures
